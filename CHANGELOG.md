# Changelog

## 0.1.0-alpha1
- Initial release: streaming pipeline core, chaining, cancellation, adapters (map/filter/inspect), basic sources/sinks and tests.

## 0.1.0-alpha2
- Added support for custom error handling and retry mechanisms.
- Improved performance optimizations for large-scale data processing.
- Enhanced documentation with detailed examples and usage guides.
- Introduced new adapters for common data transformations and filtering operations.
- Expanded test suite with more comprehensive coverage and edge case scenarios.
